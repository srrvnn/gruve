GRUVE Issues

Footpaths in GRUVE map

Our map still has references to pedestrian footpaths which are not supported on Google Streetview. The problem therefore is that the system might generate instructions involving footpaths that are not possible to reach using Google Streetview. We are working on removing those references.

More Gameworlds

Currently, there is only one gameworld available for use. We hope to provide more gameworlds with new plots in new neighbourhoods very soon.
