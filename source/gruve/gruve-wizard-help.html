GRUVE Wizard of Oz - Help

How to set up the WoZ study?

Step 1:
The GRUVE User interface (gruve-user.jsp) must be set to interact with the wizard as below in the webserver.

//Set buddy as wizard if you want the user to interact with a wizard instead of the system
//buddy = "system";
buddy = "wizard";


Step 2: Load the GRUVE wizard interface (wizards-desk.html) on the wizard's desktop. This tool will be in stand-by mode awaiting inputs from the gruve-user.jsp tool.

Step 3: Load the GRUVE User interface (gruve-user.jsp) tool on the user's desktop.

How does the wizard's desk work?

The wizard's desk will shortly receive updates about user's location from the GRUVE user interface tool. This will reflect on Streetview panel and the map on the wizard's desk page. As the user starts walking, his/her location and bearing information will get constantly updated at the wizard's desk. When the user interacts with his buddy using the interaction panel, the corresponding dialogue act will appear on the wizard's desk. The wizard can respond to the user by typing into the text box and pressing the SEND button.

How can wizards find directions to destinations?

The wizard can get directions from the user's current location to his destination by selecting his destination and pressing the "Get Directions" button. The directions will be plotted on the map, which can be used to instruct the user.

Where are the logs?

The interaction between the user and the wizard will be logged at "gruve/ilogs". These logs are prefixed as "woz-".
